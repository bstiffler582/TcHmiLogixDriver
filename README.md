## TcHmiLogixDriver

A Rockwell Automation communcation driver for TwinCAT HMI.

Uses the [libplctag.NET](https://github.com/libplctag/libplctag.NET/) library for EtherNet/IP communication to ControlLogix platform PLCs.

## Basic Installation
To install the package only within TwinCAT HMI download the latest version `.nupkg` from the [`/release`](https://github.com/bstiffler582/TcHmiLogixDriver/tree/master/release) folder. Then place the file in `C:\ProgramData\Beckhoff\NuGetPackages`. After copying the file, it will be availabe in your TwinCAT HMI project under `References\Manage NuGet Packages`.
